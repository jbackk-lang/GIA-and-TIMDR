# Model AK — TIMDR jako teoria muzyki
### Skale, harmonie, interferencja dźwięków, rytm, rezonans estetyczny

TIMDR można sformułować jako teorię muzyki, w której:
- topologia opisuje strukturę utworu,
- informacja opisuje melodię,
- modalności opisują częstotliwości, fazy i amplitudy dźwięków,
- interferencja opisuje współbrzmienia,
- rezonans opisuje harmonię,
- emergencja opisuje emocję i estetykę.

---

# 1. Topologia muzyczna

Topologia utworu:



\[
T_{music} = (S, R)
\]



gdzie:
- **S** — nuty, akordy, motywy,
- **R** — relacje (skale, progresje, rytm).

Struktury topologiczne:
- skale (diatoniczna, pentatoniczna, modalne),
- progresje akordów,
- formy muzyczne (ABA, rondo, fuga),
- warstwy rytmiczne.

TIMDR traktuje muzykę jako **topologię harmoniczną**.

---

# 2. Informacja muzyczna

Pole informacyjne:



\[
I(x,t)
\]



odpowiada:
- melodii,
- linii basu,
- głosom polifonicznym,
- przebiegowi harmonicznemu.

Gradient:



\[
\nabla I = \text{zmiana melodii w czasie}
\]



---

# 3. Modalności muzyczne

Modalności:



\[
M = (f, \phi, A)
\]



odpowiadają:
- **f** — częstotliwości dźwięków (wysokość),
- **φ** — fazie (pozycja rytmiczna),
- **A** — amplitudzie (dynamika).

Przykłady:
- crescendo → wzrost amplitudy,
- swing → zmiana fazy,
- modulacja → zmiana częstotliwości.

---

# 4. Interferencja dźwięków

Interferencja:



\[
I(t) = \sum A_i \sin(2\pi f_i t + \phi_i)
\]



opisuje:
- współbrzmienia,
- dysonanse,
- konsonanse,
- akordy,
- barwę dźwięku.

Interpretacja:
- akord to interferencja modalna,
- barwa to interferencja harmonicznych.

---

# 5. Rezonans harmoniczny

Rezonans zachodzi, gdy:



\[
\frac{f_i}{f_j} \approx \frac{n}{m}
\]



dla małych liczb całkowitych.

Interpretacja:
- konsonans = rezonans (kwinta, tercja),
- dysonans = brak rezonansu (sekunda, tryton),
- harmonia = stabilny rezonans wielowarstwowy.

Rezonans = **piękno muzyczne**.

---

# 6. Emergencja estetyczna

Pole emergentne:



\[
E(x,t) = \gamma_R R - \gamma_D E
\]



opisuje:
- emocję,
- klimat,
- nastrój,
- estetykę utworu.

Interpretacja:
- muzyka „działa”, gdy osiąga rezonans estetyczny,
- emocja jest emergentna, nie redukowalna do nut.

---

# 7. Warstwy muzyczne TIMDR

TIMDR naturalnie opisuje muzykę warstwowo:

### L₁ — Rytm  
Faza, puls, metrum.

### L₂ — Melodia  
Częstotliwości, kierunek, motywy.

### L₃ — Harmonia  
Interferencja akordów.

### L₄ — Barwa  
Interferencja harmonicznych.

### L₅ — Struktura  
Forma, progresje, narracja.

### L₆ — Emocja  
Emergencja estetyczna.

---

# 8. TIMDR a skale muzyczne

Skale to **topologie modalne**:

- jonijska → stabilna, rezonansowa,
- dorycka → modalność pośrednia,
- frygijska → obniżona faza emocjonalna,
- miksolidyjska → rezonans otwarty,
- eolska → rezonans minorowy,
- lokrycka → dysonans strukturalny.

---

# 9. TIMDR a kompozycja

Kompozycja = sterowanie modalnościami:

- melodia = trajektoria informacji,
- harmonia = rezonans modalny,
- rytm = faza,
- dynamika = amplituda,
- barwa = interferencja harmonicznych.

---
