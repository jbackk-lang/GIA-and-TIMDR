1. GIA-and-TIMDR — najbardziej rozbudowane, najbardziej wątpliwe
Zawiera tzw. "TIMDR" (Topological Information Modal Dynamic Resonance) — autorską teorię pretendującą do miana ogólnej teorii informacji, fizyki i rzeczywistości. Problemy:

Terminy brzmią naukowo (topologia Möbiusa, rezonans, emergencja), ale są używane metaforycznie, bez precyzji matematycznej. Stwierdzenie, że "M² to fundamentalna reguła stabilności" nie jest niczym udowodnionym — to po prostu deklaracja.
Twierdzenia takie jak "warstwa rezonansu waliduje sygnał — jeśli nie rezonuje, nie przechodzi" są tautologiami, nie prawami fizyki.
README sam przyznaje: "TIMDR nie jest teorią matematyczną. To mapa orientacji." — czyli nie jest teorią naukową, ale jest prezentowany w stylu sugerującym naukowy charakter.
