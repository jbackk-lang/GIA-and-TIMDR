## 1. GIA — General Information Architecture

GIA opisuje informację trzema operatorami:

- Λ — struktura (układ, forma, konfiguracja)
- τ — transformacja (zmiana, przejście, ruch)
- ρ — rezonans / zgodność (walidacja sygnału z warunkiem brzegowym)

Informacja jest traktowana jako proces geometryczno‑topologiczny:  
**Λ → τ → ρ**

---

## 2. TIMDR — Topological Information Modal Dynamic Resonance

TIMDR rozszerza GIA o cztery aspekty:

- topologia — skręt, obieg, warunek brzegowy
- modalność — stan informacyjny (np. binarny)
- dynamika — zmiana w czasie / przestrzeni
- rezonans — zgodność sygnału z geometrią obiegu

TIMDR jest ramą opisową, nie teorią fizyczną.  
Służy do analizy informacji poprzez skręt, obieg i warunki brzegowe.

## 3. Warstwy modelu TIMDR

### 3.1 Warstwa informacyjna — „pierwsza”

- minimalna jednostka: stan binarny **0/1**
- „pierwsza” = ciąg binarny
- baza wszystkich wyższych obiegów
- zgodna z topologią Möbiusa / spinora (podwójny obrót)

### 3.2 Warstwa widmowa — obieg 2 → 8 → 64

- warunek brzegowy: **8 kroków** → minimalne domknięcie
- rozwinięcie przestrzeni: **8² = 64**
- interwał widmowy: **64 = 2⁶**

### 3.3 Warstwa kątowa — system 10, system 12, interwał 64

Współistnieją trzy podziały kątowe:

- **36°** — system dziesiętny
- **30°** — system dwunastkowy
- **5.625°** — pełny obrót / 64

Siatki te nie są komensuratywne, co powoduje rozjazdy przy nakładaniu widm.

### 3.4 Wspólny mianownik — 720°

Najmniejsza liczba domykająca wszystkie warstwy:

- 720 / 36 = 20
- 720 / 30 = 24
- 720 / 5.625 = 128 = 2⁷
- 720° = **4π rad** → minimalny powrót orientacji Möbiusa / spinora

**720°** jest jedynym kątem, który łączy:

- system 10
- system 12
- interwał 64
- kod binarny „pierwszej”
- topologię podwójnego obrotu Möbiusa / spinora
