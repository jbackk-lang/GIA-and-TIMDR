# Model Interferencyjny TIMDR

## 1. Definicja interferencji



\[
I(t) = \sum_{i=1}^{n} A_i \sin(2\pi f_i t + \phi_i)
\]



## 2. Własności interferencji
- nieliniowość,
- lokalne minima i maksima,
- regiony stabilne (antinodes),
- regiony niestabilne (nodes).

## 3. Znaczenie
Interferencja jest podstawowym mechanizmem organizującym strukturę.
